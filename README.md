## 🚀 快速开始

```bash
# 安装 Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 克隆本仓库
git clone https://github.com/JansenZ/rust-learning-book.git

# 阅读章节
cd rust-learning-book
```

## 📝 学习建议

1. 按顺序阅读章节，Rust 的概念层层递进
2. 每章的代码示例都要亲手敲一遍
3. 完成每章的练习题
4. 第 10 章的实战项目是综合检验
