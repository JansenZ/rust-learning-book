# 介绍

欢迎来到 Rust 学习小册！

这是一本从基础到进阶的 Rust 编程语言学习指南。

## 🎯 适合人群

- 有编程基础，想学习 Rust 的开发者
- 想深入理解系统编程的程序员
- 对内存安全和高性能感兴趣的工程师

## 🚀 快速开始

```bash
# 安装 Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 克隆本仓库
git clone https://github.com/JansenZ/rust-learning-book.git

# 阅读章节
cd rust-learning-book
```

## 📝 学习建议

1. 按顺序阅读章节，Rust 的概念层层递进
2. 每章的代码示例都要亲手敲一遍
3. 完成每章的练习题
4. 第 10 章的实战项目是综合检验
